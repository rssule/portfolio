# Comments on Visualizations
The first visualization <i>(Viz 1)</i> is a bar chart showing general government debt in 2020 across 36 countries – alongside the OECD average and OECD total values. It is an ordered bar chart showing the ascending ranks (ordered list) of values from left to right. For this visualization, color is used to highlight the OECD average value by using balck for that bar and using grey for the other bars – creating an illusion of focus on the OECD average with the other bars in the background. The second visualization <i>(Viz 2)</i> is a grid of line charts showing a changing time series of debt-to-GDP ratio for the various countries in the data. A single color is used for all the line graphs which places equal focus to each line graph i.e. color is not used to draw focus. The third visualization <i>(Viz 3)</i> is a box plot showing distribution of debt-to-GDP ratio trend for the countries in the data. A time slider is included to navigate through the years and visually follow the change in the distribution across the years. Each box plot shows the median and range of the data set for the year.  I selected a box plot for the third visualization to provide a visual summary of the distribution of debt-to-GDP ratio across the time period in the data set. The box plot quickly enables the viewer to visualize the median value, dispersion of the data set and any possible skewness in the data for each year.

The bar chart in <i>Viz 1</i> shows ranking of the data for a specific year, the grid of line charts in <i>Viz 2</i> show change of the values over time while the box plots in <i>Viz 3</i> show distribution of the values across the time period of the data. Ranking in Viz 1 highlights the important value in the ordered list in comparison to the other values, line chart in Viz 2 gives emphasis to the changing trend of the debt-to-GDP ratio over time, while the box plot in Viz 3 shows the shape of the debt-to-GDP ratio distribution for each year in the data set.

Data Source: OECD (2022), General government debt (indicator). doi: 10.1787/a0528cc2-en (Accessed on 10 September 2022)
